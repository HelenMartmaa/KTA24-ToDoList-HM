# KTA24-ToDoList-HM

This is To-do list app with backend and frontend:
- SQLite database
- Frontend with React

Functionalities:
- Add entries to To-do list
- Remove entries from To-do list
- Mark entries as done
- Automatic additional lists to display all completed tasks/entries in one and all unfinished tasks/entries in another separate lists
- Progress bar to display how much of the entered tasks/entries are completed
